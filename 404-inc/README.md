# Required programs to run project
* Node.js version 16 or greater

# Client side
## Navigate to client side

$cd 404-inc

## Install libraries

$ npm i

## Start project

$ npm start

# Server side
## Declare environment variables
* copy '.env-example' and rename it to '.env'
* fill variables in '.env' file

## Navigate to server side

$cd server

## Install libraries
$ npm i

## Launch server
$ npm start
